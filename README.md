# pironman-u1-firmware (esp32-s2)

Firmware for Pironman U1

## Compile

- Board:
  - arduino-esp32 == 2.0.15
- Libraries:
  - ArduinoJson
  - ESPAsyncWebSrv (or ESPAsyncWebServer <https://github.com/me-no-dev/ESPAsyncWebServer>)
